## 3. TIPOGRAFIA

### Fonte Principal (Títulos)
**Inter** ou **Poppins**
- Weight: 600 (Semi-bold) a 700 (Bold)
- Uso: Headlines, títulos, logo text

### Fonte Secundária (Corpo)
**Inter** ou **Open Sans**
- Weight: 400 (Regular) a 500 (Medium)
- Uso: Parágrafos, descrições, UI

### Fonte Código
**JetBrains Mono** ou **Fira Code**
- Uso: Snippets de código, elementos técnicos

### Hierarquia

| Elemento | Tamanho | Weight | Line Height |
|----------|---------|--------|-------------|
| H1 | 48px / 3rem | 700 | 1.2 |
| H2 | 36px / 2.25rem | 600 | 1.25 |
| H3 | 24px / 1.5rem | 600 | 1.3 |
| Body | 16px / 1rem | 400 | 1.6 |
| Small | 14px / 0.875rem | 400 | 1.5 |

## 5. ELEMENTOS VISUAIS

### Ícones
- Estilo: Outline ou filled minimalista
- Stroke: 1.5px - 2px
- Cores: Seguir paleta principal

### Formas
- Nodes conectados (representa rede/conhecimento)
- Formas moleculares/científicas
- Elementos de código (brackets, nodes)

### Fotos
- Usar overlay de gradiente da marca
- Preferir imagens com tons frios (azul, roxo)
- Evitar imagens muito saturadas

---

## 6. TOM DE VOZ

### Personalidade
- **Acessível**: Linguagem clara, sem jargões desnecessários
- **Inovador**: Focado em tecnologias atuais e futuras
- **Prático**: Ênfase em aprendizado hands-on
- **Inspirador**: Motivar a transformação de carreira

### Exemplos
- "Aprenda fazendo, evolua experimentando"
- "Seu laboratório de tecnologia"
- "Do zero ao deploy"
